Module Contents

.. py:class:: ConvertFileFormat

   .. py:method:: to_numpy(tensor)
      :staticmethod:


      Converts a PyTorch tensor to a NumPy array.

      :param tensor: The tensor to be converted.
      :type tensor: torch.Tensor or numpy.ndarray

      :returns:

                The converted NumPy array if the input is a PyTorch tensor.
                            If the input is already a NumPy array, it is returned unchanged.
      :rtype: numpy.ndarray

      :raises ValueError: If the input is neither a PyTorch tensor nor a NumPy array.



   .. py:method:: torch_var_to_numpy(tensor_var)
      :staticmethod:


      Converts a Pytorch tensor to a NumPy array with dimensions rearrangement.

      :param tensor_var: A Pytorch tensor. Normally a batch of 3D (Ch x Height x Width) images in the format (Batch, Channels, Height, Width).
      :type tensor_var: torch.Tensor

      :returns: The tensor converted to a NumPy array with dimensions rearranged to (Batch, Height, Width, Channels).
      :rtype: numpy.ndarray



   .. py:method:: numpy_to_torch_var(np_tensor, device)
      :staticmethod:


      Converts a NumPy array to a PyTorch tensor with appropriate dimension rearrangement.

      :param np_tensor: The NumPy array representing the tensor
      :type np_tensor: numpy.ndarray
      :param device: The device where the tensor will be stored.
      :type device: torch.device

      :returns:

                The equivalent Pytorch tensor with dimensions formatted as
                              (Batch, Channels, Height, Width).
      :rtype: torch.Tensor



   .. py:method:: permute_tensor_dims(input_var)
      :staticmethod:


      Rearranges the dimensions of a NumPy array for PyTorch compatibility.
          H x W x C -> C x H x W.

      :param input_var: The NumPy array in the format (H, W, C).
      :type input_var: numpy.ndarray

      :returns: A Pytorch tensor in the format (C, H, W).
      :rtype: torch.Tensor



   .. py:method:: to_torch(ndarray)
      :staticmethod:


      Converts a NumPy array to a PyTorch tensor if it is not already a tensor.

      :param ndarray: The input data to be converted to a PyTorch tensor.
      :type ndarray: numpy.ndarray or torch.Tensor

      :returns: A PyTorch tensor. If the input is already a tensor, it is returned as is.
      :rtype: torch.Tensor

      :raises ValueError: If the input is neither a NumPy array nor a PyTorch tensor.



   .. py:method:: im_to_numpy(img)
      :staticmethod:


      Converts an image from PyTorch tensor to a NumPy array in the standard image format.

      :param img: The image tensor or array.
                  If it is a tensor, it must be in the format (C, H, W).
      :type img: torch.Tensor or numpy.ndarray

      :returns: The image as a NumPy array in the format (H, W, C).
      :rtype: numpy.ndarray



   .. py:method:: im_to_torch(img)
      :staticmethod:


      Converts an image from OpenCV/Numpy format (H, W, C) to PyTorch format (C, H, W).

      :param img: The image as a NumPy array with the format (H, W, C).
      :type img: numpy.ndarray

      :returns: The image as a PyTorch tensor with the format (C, H, W).
      :rtype: torch.Tensor



   .. py:method:: torch_to_img(img)
      :staticmethod:


      Converts a PyTorch tensor to a NumPy array in image format.

      :param img: The image tensor, typically in the format (C, H, W).
      :type img: torch.Tensor

      :returns: The image as a NumPy array in the format (H, W, C).
      :rtype: numpy.ndarray
